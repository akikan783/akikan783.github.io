Production

作ったもの

やってみよう！短文入力

  概要

このアプリケーションはユーザーのタイピング能力を鍛えるアプリです。
時間を計測しながら約40文字程度の短文を連続して入力することでタイピング速度の向上、正誤判定機能により正確に入力できたかわかるようになっています。
目標設定しやすいように2つの計測モードを用意しています。
10分間で何問できるかを計測する「10分間計測モード」、10問にかかる時間を計測する「10問計測モード」が選べるようになっています。
制作期間 : 2018年12月～2019年3月頃

  制作環境

  • 言語 : Java
  • IDE : Eclipse
  • ライブラリ : JavaFX
  • ツール : Scene Builder

  作った目的

Javaの勉強が一区切りついたので、おさらい・実力試し目的でアプリを作ろうと考えた。
開発環境を整えるのが容易なGUIアプリが作りやすいと考えた。
JavaでGUIアプリを作る場合のライブラリであるJavaFXの使い方を理解したかった。
ゲーム形式のアプリが作りたく、入力に対する正誤判定と結果の出力が分かりやすい今回の形となった。
